 DSO.hThe Digital Signal Oscillator (DSO) class and instructions. The AS3397 chip has an external input just before its analog filter, where the chip's analog oscillator is mixed with a digital signal generated by the RP2040 microcontroller (see documentation for more technical details). Currently, the DSO functions as a relatively simple wavetable synthesizer, but it could be easily extended. The wavetable synth can add two shifted wavetable to create interesting chorus-like effect
